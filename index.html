Birthday Calculator

This web aplication is created so that it can be able to calculate the actual day of the week when a user was born and displays the Akan name(Ghanaian name given as per the day of the week one was born).It provides a form which includes the year, month and day when a user was born. As the user keys in the said details of his/her birthday and clicks submit, their Akan name is displayed. If a wrong information is keyed in then an invalid message displays.
